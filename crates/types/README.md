## signet-storage-types

Primitive types for use in the Signet storage system.

These types are _generally_ migrated from `reth`, but have been adapted to fit
the needs of Signet. They are designed to be efficient, easy to use, and
compatible with the Signet storage architecture.

# Provenance

Significant portions of this code were originally developed for the
[Reth](https://github.com/paradigmxyz/reth) project by Paradigm. They have been
adapted and extended for use in the Signet project, with contributions from the
Signet development team.
